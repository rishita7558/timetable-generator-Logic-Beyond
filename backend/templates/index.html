Quick Start Guide

2
Generate Timetables

3
Generate Exam Schedules

Navigate to "Generate Exam Schedule" to create conflict-free exam timetables

4
View & Download

Use filters to view specific timetables and download individual or all schedules

CSV File Requirements

course_data.csv

Required columns: Course Code, Course Name, Semester, Department, LTPSC, Credits, Instructor, Elective (Yes/No)

faculty_availability.csv

Required columns: faculty name, Available Days, Unavailable Time Slots

exams_data.csv

Required columns: Course Code, Course Name, Department, Semester, Credits, Exam Type, Duration, Preferred Date

Other Files

classroom_data.csv, student_data.csv with appropriate columns

Troubleshooting

No timetables generated?

Check that all required CSV files are uploaded and have the correct format

No exam schedules generated?

Ensure exams_data.csv is uploaded and contains valid exam data with preferred dates

Elective courses not showing?

Ensure elective courses are marked "Yes" in the Elective column

Exam scheduling conflicts?

Try increasing the exam period duration or allowing more exams per day

Use the debug endpoints to check loaded data and file matching
